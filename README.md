# Scribble
A simple digit recognizer using pygame in which you can draw the digits and recognize it straight away using our own CNN model.

# Dependencies
```bash
$ pip3 install -r requirements.txt
```

# Instructions
```bash
$ python3 Scribble/scribble_main.py
```
